Working with mongoose and mongodb

Schema--->>> Define the structure using Schema
Schema--->>> Using Schema we create a model
Model--->>> Using a model we do CRUD operations

Schema✅
const userSchema = new mongoose.Schema({
  firstName: {
    type: String,
    required: true
  },
  lastName: {
    type: String,
  },
  email: {
    type: String,
    required: true,
    unique: true,
  },
  jobTitle: {
    type: String,
  },
  gender: {
    type: String,
  }
})

Model✅
const userCollection = mongoose.model('user', userSchema)
